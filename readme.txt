Loads Twitter and Instagram posts using 2 methods: search by tag and timeline feed. 

== Description ==

Loads Twitter and Instagram posts. For each source, the plugin can perform a tag search or load from one or more accounts' timeline. 
Multiple accounts and tags can be specified. The posts are saved in database and data can be loaded from theme or other plugins. Exclude words filter is available.

== Installation ==

1. Upload `socialfeed` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In Settings page configure the tags, the accounts and excluded words 
4. Manually trigger the fetch or set a cron with the url provided in Settings page
5. Place the code from Settings page in template to load data
